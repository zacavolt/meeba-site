Création d'un serveur web

Lors de cette activité pratique (AP), nous avons appris à configurer un serveur web sous Linux en utilisant Apache2, PHP, et MariaDB pour gérer une base de données MySQL. Tout d'abord, nous avons installé et configuré Apache2, un serveur HTTP, pour qu'il puisse héberger nos sites web. Nous avons ensuite installé PHP pour permettre l'exécution de scripts dynamiques sur nos pages, rendant ainsi notre serveur capable de gérer des applications web interactives. Par la suite, nous avons configuré MariaDB, une solution de gestion de base de données compatible avec MySQL, pour stocker et gérer les données nécessaires à nos applications.

Après avoir mis en place les services de base, nous avons appris à sécuriser notre serveur et à optimiser son fonctionnement. Nous avons configuré des paramètres de sécurité pour éviter les accès non autorisés et ajusté les configurations d'Apache2 et PHP pour garantir la stabilité et les performances. Enfin, nous avons testé le serveur en y déployant une application web simple, connectée à la base de données MariaDB, afin de vérifier que toutes les parties du système fonctionnaient correctement ensemble. Cette activité nous a permis de comprendre les étapes essentielles pour déployer un serveur web complet en environnement Linux.
